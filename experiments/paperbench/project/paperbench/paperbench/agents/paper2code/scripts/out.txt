[Planning] Overall plan
============================================
Below is a detailed roadmap outlining how to reproduce the experiments and methodologies described in the paper. This plan is organized into major sections that mirror the paper’s structure—from building a toxicity probe and extracting toxic vectors, to constructing a pairwise dataset and fine-tuning with DPO, and finally performing mechanistic and “un‐alignment” experiments. Each step is described with implementation details, datasets, hyperparameter hints, evaluation metrics, and notes on potential ambiguities.

──────────────────────────────
1. Environment and Setup

• Language and Frameworks
 – Use Python (≥3.8) with a deep learning framework (e.g., PyTorch).
 – Leverage the HuggingFace Transformers library for model loading and inference.

• Hardware
 – Use GPUs for training and fine-tuning large models (e.g., GPT2-medium and Llama2-7b).

• Repositories and Data
 – Download pre-trained GPT2-medium and Llama2-7b models.
 – Acquire the REALTOXICITYPROMPTS dataset (specifically, the “challenge” subset of 1,199 prompts).
 – Obtain Wikitext-2 (for perplexity evaluation) and a Wikipedia continuation dataset (≈2,000 sentences) for F1 evaluation.
 – (Optional) Identify, if necessary, an external toxicity rating tool (e.g., Perspective API or a comparable substitute) to score outputs.

• Code Modularity
 – Write separate modules for:
  a. Toxicity probe training and evaluation.
  b. Extraction and SVD analysis of toxic vectors.
  c. Intervention routines during generation.
  d. Pairwise dataset generation via PPLM.
  e. DPO fine-tuning.
  f. Post-DPO mechanistic analyses and “un-alignment” experiments.
──────────────────────────────
2. Toxicity Probe & Toxic Vector Extraction (Methodology, Section 3)

A. Training a Toxicity Probe
 • Objective: Learn a linear classifier (W_Toxic) on the final layer’s residual stream (xₗ₋₁) that predicts whether a sample is toxic.
 • Procedure:
  – For each input sequence, run a forward pass and extract the hidden state at the final residual stream (after the last transformer layer, averaged across timesteps).
  – Use a simple linear layer with softmax (P(toxic|xₗ₋₁) = softmax(W_Toxic × xₗ₋₁)) to classify examples.
  – Use a 90:10 train–validation split. Ensure the probe reaches an accuracy of ~94% on validation.
 • Data Source:
  – Source toxic and non-toxic examples. (If not explicitly defined, use a known toxicity dataset or sample from outputs using Perspective API thresholds.)
 • Note: Exactly how the labeled examples are collected is not fully detailed; if unclear, document the strategy or try multiple thresholds.

B. Extracting Toxic Vectors from MLP Blocks
 • For each pre-trained model (GPT2 and Llama2):
  – Identify and hook into the MLP modules (and for Llama2, the GLU structure) across layers.
  – From each MLP block, extract:
   ○ Value vectors (vᵢ) and corresponding key vectors (kᵢ). (For Llama2, also separately track the gating component, σ(W₁x), and W₂).
 • Determine “toxicity” of each value vector by computing the cosine similarity between vᵢ and the trained toxic probe weight (W_Toxic).
  – Select those with the highest similarity as your toxic set, naming them MLP.v_Toxic and storing their corresponding MLP.k_Toxic.
 • SVD Analysis:
  – Stack N (e.g., N = 128 as in the paper) toxic value vectors into a matrix.
  – Perform singular value decomposition (SVD) to obtain basis vectors (SVD.U_Toxic).
  – Use these basis vectors later to inspect which tokens are promoted (by projecting onto the vocabulary space via the unembedding matrix E) and compare with paper tables.
 • Visualization:
  – For sanity checking, project each toxic vector into the vocabulary space to extract top tokens (mimicking Table 1). Document these results.
──────────────────────────────
3. Intervention Experiments on Pre-trained Models (Section 3.3)

A. Intervention Procedure
 • Implement a function that modifies the forward pass:
  – At the final residual stream (xₗ₋₁), subtract a scaled toxic vector: xₗ₋₁_new = xₗ₋₁ − α × W.
  – The vector W can be the probe vector (W_Toxic), a particular MLP.v_Toxic, or one of the SVD.U_Toxic basis vectors.
 • Choose a heuristic scale factor α such that the output perplexity remains similar to that of a post-alignment (DPO) model.

B. Experimental Setup
 • Use the “challenge” subset of REALTOXICITYPROMPTS (1,199 prompts).
  – For each prompt, generate 20 tokens using the baseline model.
  – Run parallel generations (with and without the intervention).
 • Record Metrics:
  – Toxicity: Use the Perspective API (or equivalent) to rate generated text.
  – Perplexity: Evaluate on Wikitext-2.
  – F1 Score: Using ~2,000 Wikipedia sentences, compute the harmonic mean of precision and recall where precision is the fraction of generated tokens present in the ground truth continuation and recall is the fraction of ground truth tokens in the generated text.
 • Expected Outcome:
  – A drop in toxicity scores when subtracting toxic vectors, with minimal impact on perplexity and F1.
──────────────────────────────
4. Constructing the Pairwise Toxic Dataset & DPO Fine-Tuning (Sections 4 and 4.2)

A. Constructing Pairwise Data using PPLM
 • Data Generation:
  – Use sentences from Wikitext-2 as prompts.
  – For each prompt, generate two continuations:
   ○ Positive (preferred, non-toxic): Use the base GPT2 with greedy sampling.
   ○ Negative (non-preferred, toxic): Use PPLM to steer generation toward toxicity. In this process:
    – Incorporate the toxic probe (W_Toxic) as the attribute classifier.
    – Adjust PPLM’s hyperparameters (step size, number of iterations, etc.). (Note: exact hyperparameters are in Appendix E of the paper; if they are not available, document the values you try.)
  – Aim to create around 24,576 pairs; note that early stopping during DPO training may occur after ≈6,700 sample pairs.
 • Save the paired examples with prompt, positive continuation, and negative (toxic) continuation.

B. Implementing DPO Fine-Tuning
 • DPO Loss Function:
  – The loss is defined as:  
   L_DPO = – E[ log σ(β (log P – log N)) ]  
   where:
    P = π_θ(y⁺|w) / π_ref(y⁺|w)  
    N = π_θ(y⁻|w) / π_ref(y⁻|w)  
   and β is a hyperparameter scaling constant.
 • Model Setup:
  – Freeze a copy of the original (pre-trained) model as π_ref.
  – Fine-tune another copy (π_θ) using the DPO loss on the pairwise dataset.
 • Optimization:
  – Choose an optimizer (e.g., Adam) with a learning rate determined via preliminary experiments. (If Appendix E hyperparameters are not available, start with a common value like 1e-5 or 1e-4.)
  – Use early stopping with a patience value of 10. Monitor validation loss (using a held-out part of the pairwise data).
 • Output:
  – Save the resulting fine-tuned models (label as GPT2_DPO and Llama2_DPO).
──────────────────────────────
5. Post-DPO Evaluation and Mechanistic Analysis (Sections 5 and 5.2)

A. Evaluation Metrics
 • Toxicity: Again, generate outputs from the challenge prompts and measure toxicity with the Perspective API.
 • Perplexity: Use Wikitext-2.
 • F1: Evaluate on 2,000 Wikipedia sentences.

B. Analyzing Changes in Latent Representations
 • Compare the pre-DPO and post-DPO models:
  – Extract the residual stream from an intermediate layer (e.g., x_mid after attention but before the MLP).
  – Compute the difference vector δx = x_mid_DPO − x_mid_pre.
 • Analyze “Bypass” Behavior:
  – For GPT2_DPO, visualize how the residual stream shifts away from the toxic regions defined by the MLP.k_Toxic.
  – Compute the cosine similarity between δx and the changes in the toxic value vectors (δMLP.v) from preceding layers.
  – Create plots similar to Figures 2–5 in the paper.
 • Llama2_DPO Specifics:
  – Instead of a shift in residual stream, inspect the gating components (σ(W₁x)) and the contribution of W₂. Visualize that these components “turn off” toxic vectors.
  – Compare activations before and after DPO fine-tuning (mimicking Figure 6).

C. Documentation:

 1. Loop over epochs:
  a. Shuffle the training data (the list of triplets) at the beginning of each epoch.
  b. Process the training set in batches of size specified by config.training.batch_size.
  c. For every batch, extract three lists (or tensors) of data:
   – prompts (w)
   – positive continuations (y⁺)
   – negative continuations (y⁻)

• Log-probability Computation:
 For each triplet (sample) in a batch, compute the following:
  a. Use the student model (πθ) to compute the log probability of the positive continuation given the prompt. This can be done by passing the concatenated sequence [prompt, y⁺] through the model and computing the sum of log probabilities for the tokens in y⁺.
  b. Similarly, compute the log probability of y⁺ under the reference model (π_ref).

   – log πθ(y⁻|w)
   – log π_ref(y⁻|w)

  – log_ratio_positive = log πθ(y⁺|w) − log π_ref(y⁺|w)

  loss_sample = – log(σ(delta))
  – To ensure numerical stability, use a “logsigmoid” function instead of log(sigmoid(x)).
 4. Aggregate this loss over the entire batch (e.g., by averaging).

 – Call backward() on the computed loss.

"""model.py

This module contains the following classes:
  - TransformerModel: A wrapper for HuggingFace pre-trained language models (e.g., GPT2-medium, Llama2-7b).
    It loads the model, registers forward hooks to capture intermediate activations (e.g., x_mid),
    and implements a forward() method.
    
  - ToxicityProbe: A PyTorch module implementing a simple linear classifier that computes 
    P(Toxic | x_{L-1}) = softmax(W_Toxic * x_{L-1}). It includes methods for forward propagation 
    and prediction, and provides access to the toxic class weight vector.
    
  - MLPExtractor: A utility class that extracts internal MLP block vectors (key and value vectors)
    from the TransformerModel. It computes cosine similarities with a provided toxicity probe’s weight 
    (W_Toxic) to select toxic vectors, and offers an SVD decomposition method for further analysis.
    
All classes use explicit types and default values where necessary. Configuration values (such as
the pretrained model name) are expected to be provided via the configuration file (config.yaml).
"""

import logging
from typing import Any, List, Tuple, Optional, Dict

import torch
import torch.nn as nn
import torch.nn.functional as F
import numpy as np
from transformers import AutoModelForCausalLM, AutoTokenizer


class TransformerModel:
    """
    TransformerModel wraps a HuggingFace pre-trained language model and registers hooks to capture
    intermediate activations (e.g., x_mid from each transformer block's MLP input).
    
    Attributes:
        pretrained_model_name (str): Name of the pre-trained model (e.g., 'gpt2-medium' or 'llama2-7b').
        model (Any): The loaded HuggingFace pre-trained language model.
        activations (Dict[str, torch.Tensor]): Dictionary storing the intermediate activations captured by hooks.
        hook_handles (List[torch.utils.hooks.RemovableHandle]): List storing hook handles for removal if needed.
        device (torch.device): Device on which the model is loaded.
    """

    def __init__(self, pretrained_model: str = "gpt2-medium") -> None:
        """
        Initialize the TransformerModel.
        
        Args:
            pretrained_model (str): The identifier for the HuggingFace pre-trained model.
                Defaults to "gpt2-medium".
        """
        self.pretrained_model_name: str = pretrained_model
        self.logger: logging.Logger = logging.getLogger(__name__)
        self.device: torch.device = torch.device("cuda" if torch.cuda.is_available() else "cpu")
        self.logger.info(f"Loading pre-trained model: {self.pretrained_model_name}")
        self.model: Any = AutoModelForCausalLM.from_pretrained(self.pretrained_model_name)
        self.model.to(self.device)
        self.model.eval()
        # Dictionary to store intermediate activations from hooks.
        self.activations: Dict[str, torch.Tensor] = {}
        # List to store hook handles.
        self.hook_handles: List[torch.utils.hooks.RemovableHandle] = []
        self.register_hooks()

    def register_hooks(self) -> None:
        """
        Register forward hooks on each transformer block to capture the x_mid activations (input to the MLP block).
        
        For GPT2, hooks are registered on block.mlp of each block in self.model.transformer.h.
        For Llama2, hooks are registered on block.mlp in self.model.model.layers.
        """
        # Determine the underlying blocks based on model type.
        blocks: Any = None
        model_name_lower: str = self.pretrained_model_name.lower()
        if "gpt2" in model_name_lower:
            # For GPT2, transformer blocks are in self.model.transformer.h
            blocks = self.model.transformer.h
            self.logger.info("Detected GPT2 model architecture for hook registration.")
        elif "llama" in model_name_lower:
            # For Llama2, transformer blocks are assumed to be in self.model.model.layers
            if hasattr(self.model, "model") and hasattr(self.model.model, "layers"):
                blocks = self.model.model.layers
                self.logger.info("Detected Llama2 model architecture for hook registration.")
            else:
                error_msg: str = "Llama2 architecture not recognized: unable to find model.model.layers."
                self.logger.error(error_msg)
                raise ValueError(error_msg)
        else:
            error_msg = f"Unsupported model type for hook registration: {self.pretrained_model_name}"
            self.logger.error(error_msg)
            raise ValueError(error_msg)

        # Register hooks on the MLP submodule of each block.
        for idx, block in enumerate(blocks):
            if hasattr(block, "mlp"):
                layer_key: str = f"layer_{idx}"
                # Define the hook function capturing the input to the MLP (x_mid)
                def hook_fn(module: nn.Module, inputs: Tuple[torch.Tensor], output: torch.Tensor, layer_key=layer_key) -> None:
                    # Capture the first element of the inputs and detach from the graph.
                    self.activations[layer_key] = inputs[0].detach().cpu()
                handle = block.mlp.register_forward_hook(hook_fn)
                self.hook_handles.append(handle)
                self.logger.info(f"Registered hook for {layer_key} on module: {block.mlp}")
            else:
                self.logger.warning(f"Block {idx} has no 'mlp' attribute; hook not registered.")

            input_ids (torch.Tensor): Tensor of token IDs.
            **kwargs: Additional keyword arguments for the model's forward() method.
            
        Returns:
            Any: The output from the model's forward() method.
        """
        # Clear previous activations.
        self.activations.clear()
        outputs: Any = self.model(input_ids=input_ids.to(self.device), **kwargs)
        return outputs


class ToxicityProbe(nn.Module):
    """
    ToxicityProbe implements a simple linear classifier (a single linear layer with softmax) on top of the
    last-layer residual stream (x_{L-1}) to predict toxicity.
    
    Attributes:
        linear (nn.Linear): Linear layer mapping the residual stream to the number of classes.
        toxic_class_index (int): Index corresponding to the toxic class.
    """
    
    def __init__(self, input_dim: int, output_dim: int = 2) -> None:
        """
        Initialize the ToxicityProbe.
        
        Args:
            input_dim (int): Dimension of the input features (d).
            output_dim (int): Number of output classes. Defaults to 2 (toxic, non-toxic).
        """
        super(ToxicityProbe, self).__init__()
        self.linear: nn.Linear = nn.Linear(input_dim, output_dim)
        # Assume the toxic class is at index 1 when there are 2 classes.
        self.toxic_class_index: int = 1 if output_dim == 2 else 0

    def forward(self, x: torch.Tensor) -> torch.Tensor:
        """
        Compute the forward pass for the toxicity probe.
        
        Args:
            x (torch.Tensor): Input tensor of shape [batch_size, input_dim].
            
        Returns:
            torch.Tensor: Softmax probabilities of shape [batch_size, output_dim].
        """
        logits: torch.Tensor = self.linear(x)
        probabilities: torch.Tensor = F.softmax(logits, dim=-1)
        return probabilities

    def predict(self, x: torch.Tensor) -> torch.Tensor:
        """
        Predict toxicity class labels given input x.
        
        Args:
            x (torch.Tensor): Input tensor.
            
        Returns:
            torch.Tensor: Predicted class labels.
        """
        probabilities: torch.Tensor = self.forward(x)
        predictions: torch.Tensor = torch.argmax(probabilities, dim=-1)
        return predictions

    def get_probe_weight(self) -> torch.Tensor:
        """
        Retrieve the weight vector corresponding to the toxic class.
        
        Returns:
            torch.Tensor: A weight vector of shape [input_dim] for the toxic class.
        """
        return self.linear.weight[self.toxic_class_index].detach()


class MLPExtractor:
    """
    MLPExtractor is a utility class for extracting internal MLP block vectors from a TransformerModel.
    It extracts value vectors and key vectors from each block's MLP module and computes their cosine
    similarity with the toxicity probe's weight to select those that promote toxicity.
    
    Methods:
        extract_toxic_vectors(top_k: int = 128) -> Tuple[List[torch.Tensor], List[torch.Tensor]]:
            Returns lists of toxic value vectors and corresponding key vectors based on cosine similarity.
            
        apply_svd(vectors: List[torch.Tensor]) -> np.ndarray:
            Performs singular value decomposition on the provided vectors and returns the U matrix.
    """

    def __init__(self, model: TransformerModel, probe: Optional[ToxicityProbe] = None) -> None:
        """
        Initialize the MLPExtractor.
        
        Args:
            model (TransformerModel): An instance of TransformerModel.
            probe (Optional[ToxicityProbe]): An optional toxicity probe for computing cosine similarities.
        """
        self.model: TransformerModel = model
        self.probe: Optional[ToxicityProbe] = probe
        self.logger: logging.Logger = logging.getLogger(__name__)

    def extract_toxic_vectors(self, top_k: int = 128) -> Tuple[List[torch.Tensor], List[torch.Tensor]]:
        """
        Extract toxic value vectors and key vectors from the model's MLP modules.
        
        For each transformer block, the method extracts:
          - For GPT2-style MLP: key vectors from c_fc.weight (each row) and value vectors from c_proj.weight (each column).
          - For Llama2-style MLP: key vectors from gate_proj.weight (each row) and value vectors from up_proj.weight (each column).
        
        Cosine similarity is computed between each value vector and the toxicity probe's weight (if provided);
        the vectors are sorted based on similarity and the top_k are selected.
        
        Args:
            top_k (int): Number of top vectors to select based on cosine similarity. Defaults to 128.
            
        Returns:
            Tuple[List[torch.Tensor], List[torch.Tensor]]: Two lists containing toxic value vectors and corresponding key vectors.
        """
        vector_list: List[Tuple[float, torch.Tensor, torch.Tensor]] = []  # Entries: (similarity, value_vector, key_vector)
        
        # Access the underlying HuggingFace model from the wrapper.
        hf_model: Any = self.model.model
        model_name_lower: str = self.model.pretrained_model_name.lower()
        blocks: Any = None
        
        if "gpt2" in model_name_lower:
            blocks = hf_model.transformer.h
            self.logger.info("Extracting MLP vectors from GPT2 architecture.")
        elif "llama" in model_name_lower:
            if hasattr(hf_model, "model") and hasattr(hf_model.model, "layers"):
                blocks = hf_model.model.layers
                self.logger.info("Extracting MLP vectors from Llama2 architecture.")
            else:
                error_msg: str = "Llama2 architecture not recognized: unable to find hf_model.model.layers."
                self.logger.error(error_msg)
                raise ValueError(error_msg)
        else:
            error_msg = f"Unsupported model type in MLPExtractor: {self.model.pretrained_model_name}"
            self.logger.error(error_msg)
            raise ValueError(error_msg)
        
        for idx, block in enumerate(blocks):
            if hasattr(block, "mlp"):
                mlp_module = block.mlp
                # GPT2-style MLP: expected attributes c_fc and c_proj.
                if hasattr(mlp_module, "c_fc") and hasattr(mlp_module, "c_proj"):
                    key_weights: torch.Tensor = mlp_module.c_fc.weight.data.clone().detach()  # Shape: [d_ff, d_model]
                    value_weights: torch.Tensor = mlp_module.c_proj.weight.data.clone().detach()  # Shape: [d_model, d_ff]
                    d_ff: int = key_weights.shape[0]
                    for i in range(d_ff):
                        key_vector: torch.Tensor = key_weights[i]  # Shape: (d_model,)
                        value_vector: torch.Tensor = value_weights[:, i]  # Shape: (d_model,)
                        similarity: float = 0.0
                        if self.probe is not None:
                            probe_weight: torch.Tensor = self.probe.get_probe_weight()
                            sim_tensor: torch.Tensor = F.cosine_similarity(value_vector.unsqueeze(0), probe_weight.unsqueeze(0))
                            similarity = sim_tensor.item()
                        vector_list.append((similarity, value_vector, key_vector))
                # Llama2-style MLP: expected attributes gate_proj and up_proj.
                elif hasattr(mlp_module, "gate_proj") and hasattr(mlp_module, "up_proj"):
                    key_weights = mlp_module.gate_proj.weight.data.clone().detach()  # Shape: [d_ff, d_model]
                    value_weights = mlp_module.up_proj.weight.data.clone().detach()  # Shape: [d_model, d_ff]
                    d_ff = key_weights.shape[0]
                    for i in range(d_ff):
                        key_vector = key_weights[i]  # Shape: (d_model,)
                        value_vector = value_weights[:, i]  # Shape: (d_model,)
                        similarity = 0.0
                        if self.probe is not None:
                            probe_weight = self.probe.get_probe_weight()
                            sim_tensor = F.cosine_similarity(value_vector.unsqueeze(0), probe_weight.unsqueeze(0))
                            similarity = sim_tensor.item()
                        vector_list.append((similarity, value_vector, key_vector))
                else:
                    self.logger.warning(f"Block {idx} mlp module does not have expected attributes; skipping.")
            else:
                self.logger.warning(f"Block {idx} does not have 'mlp' attribute; skipping.")
        
        if self.probe is not None:
            # Sort vectors by cosine similarity (highest first)
            vector_list.sort(key=lambda x: x[0], reverse=True)
            self.logger.info(f"Extracted {len(vector_list)} vectors; selecting top {top_k} based on cosine similarity.")
            selected_entries = vector_list[:top_k]
        else:
            self.logger.info("No toxicity probe provided; returning all extracted vectors.")
            selected_entries = vector_list
        
        toxic_value_vectors: List[torch.Tensor] = [entry[1] for entry in selected_entries]
        toxic_key_vectors: List[torch.Tensor] = [entry[2] for entry in selected_entries]
        return toxic_value_vectors, toxic_key_vectors

    def apply_svd(self, vectors: List[torch.Tensor]) -> np.ndarray:
        """
        Perform Singular Value Decomposition (SVD) on the provided list of vectors.
        
        Args:
            vectors (List[torch.Tensor]): List of toxic value vectors (each of shape [d_model]).
            
        Returns:
            np.ndarray: The U matrix (basis vectors) from the SVD decomposition.
        """
        if not vectors:
            self.logger.error("Empty vector list provided for SVD.")
            raise ValueError("No vectors provided for SVD.")
        # Stack vectors into a matrix of shape (N, d_model)
        matrix: torch.Tensor = torch.stack(vectors, dim=0)
        matrix_np: np.ndarray = matrix.cpu().numpy()
        U, s, Vh = np.linalg.svd(matrix_np, full_matrices=False)
        self.logger.info(f"Applied SVD on matrix with shape {matrix_np.shape}; U matrix shape: {U.shape}.")
        return U

"""trainer.py

This module implements the Trainer class for Direct Preference Optimization (DPO) fine‐tuning.
The Trainer uses a pairwise toxicity dataset (triplets of prompt, positive/non‐toxic continuation,
and negative/toxic continuation) to update the student model (πθ) with the DPO loss:

    L_DPO = – log( σ( β * ([log πθ(y⁺|w) – log π_ref(y⁺|w)] – [log πθ(y⁻|w) – log π_ref(y⁻|w)] ) ) )

A frozen copy of the pre‐trained model is created as the reference model (π_ref). Training
proceeds over mini‐batches with early stopping based on validation loss. All hyperparameters,
data splits, and saving options are read directly from the provided configuration.

Author: [Your Name]
Date: [Today's Date]
"""

import os
import random
import logging
from copy import deepcopy
from typing import List, Tuple, Dict

import torch
import torch.nn as nn
import torch.optim as optim
import torch.nn.functional as F
from transformers import AutoTokenizer

# Import the model wrapper from model.py
from model import TransformerModel


class Trainer:
    """
    Trainer implements the training loop for DPO fine-tuning of a language model.

    Attributes:
        model (TransformerModel): The student model (πθ) being fine-tuned.
        ref_model (TransformerModel): The frozen reference model (π_ref) for baseline probabilities.
        pairwise_data (List[Tuple[str, str, str]]): List of triplets (prompt, positive continuation, negative continuation).
        train_data (List[Tuple[str, str, str]]): Training subset of the pairwise data.
        val_data (List[Tuple[str, str, str]]): Validation subset of the pairwise data.
        config (Dict): Configuration dictionary with hyperparameters and settings.
        tokenizer (AutoTokenizer): Tokenizer corresponding to the pretrained model.
        device (torch.device): Device on which training occurs.
        optimizer (torch.optim.Optimizer): Adam optimizer on the student model's parameters.
        batch_size (int): Batch size for training.
        epochs (int): Maximum number of training epochs.
        early_stop_patience (int): Early stopping patience.
        dpo_loss_beta (float): Scaling constant beta used in the DPO loss.
        logger (logging.Logger): Logger instance for training progress.
    """

    def __init__(self, model: TransformerModel, pairwise_data: List[Tuple[str, str, str]], config: Dict) -> None:
        """
        Initialize the Trainer by storing the pairwise data, configuration, and preparing
        both the student and reference models.

        Args:
            model (TransformerModel): The student model (πθ) to be fine-tuned.
            pairwise_data (List[Tuple[str, str, str]]): List of triplets (prompt, positive, negative).
            config (Dict): Configuration dictionary with hyperparameters.
        """
        self.logger: logging.Logger = logging.getLogger(__name__)
        config_logging_level: str = config.get("logging", {}).get("level", "INFO")
        self.logger.setLevel(getattr(logging, config_logging_level.upper(), logging.INFO))
        self.config: Dict = config
        self.pairwise_data: List[Tuple[str, str, str]] = pairwise_data

        # Create a deep copy of the input model for the reference; freeze its parameters.
        self.model: TransformerModel = model
        self.ref_model: TransformerModel = deepcopy(model)
        self.ref_model.model.eval()
        for param in self.ref_model.model.parameters():
            param.requires_grad = False

        # Set device for both models.
        self.device = torch.device("cuda" if torch.cuda.is_available() else "cpu")
        self.model.model.to(self.device)
        self.ref_model.model.to(self.device)

        # Load tokenizer based on the pretrained model name.
        pretrained_model_name: str = self.config.get("model", {}).get("pretrained_model", "gpt2-medium")
        self.tokenizer = AutoTokenizer.from_pretrained(pretrained_model_name)
        if self.tokenizer.pad_token is None:
            self.tokenizer.pad_token = self.tokenizer.eos_token

        # Split the pairwise data into training (90%) and validation (10%) sets.
        random.shuffle(self.pairwise_data)
        split_idx: int = int(0.9 * len(self.pairwise_data))
        self.train_data: List[Tuple[str, str, str]] = self.pairwise_data[:split_idx]
        self.val_data: List[Tuple[str, str, str]] = self.pairwise_data[split_idx:]
        self.logger.info(f"Data split: {len(self.train_data)} training samples, {len(self.val_data)} validation samples.")

        # Extract training hyperparameters with default values.
        training_cfg: Dict = self.config.get("training", {})
        learning_rate: float = float(training_cfg.get("learning_rate", 1e-5))
        self.batch_size: int = int(training_cfg.get("batch_size", 8))
        self.epochs: int = int(training_cfg.get("epochs", 5))
        self.early_stop_patience: int = int(training_cfg.get("early_stop_patience", 10))
        dpo_cfg: Dict = self.config.get("dpo", {})
        self.dpo_loss_beta: float = float(dpo_cfg.get("dpo_loss_beta", 1.0))

        # Initialize Adam optimizer on the student model's parameters.
        self.optimizer = optim.Adam(self.model.model.parameters(), lr=learning_rate)

        self.logger.info(
            f"Trainer initialized with learning_rate={learning_rate}, batch_size={self.batch_size}, "
            f"epochs={self.epochs}, early_stop_patience={self.early_stop_patience}, dpo_loss_beta={self.dpo_loss_beta}."
        )

    def _compute_log_prob(self, prompt: str, continuation: str, current_model: TransformerModel) -> torch.Tensor:
        """
        Compute the log probability of the continuation given the prompt using the specified model.

        This is achieved by:
            - Concatenating the prompt and continuation.
            - Tokenizing the combined text.
            - Computing the log probabilities from the model's logits.
            - Summing the log probabilities corresponding to the continuation tokens only.

        Args:
            prompt (str): The prompt text.
            continuation (str): The continuation text.
            current_model (TransformerModel): The model (student or reference) used for probability computation.

        Returns:
            torch.Tensor: A scalar tensor representing the sum of log probabilities for the continuation.
        """
        # Concatenate prompt and continuation with a space.
        full_text: str = prompt.strip() + " " + continuation.strip()

        # Tokenize the full sequence; do not add special tokens.
        encoded_full: List[int] = self.tokenizer.encode(full_text, add_special_tokens=False)
        if not encoded_full:
            return torch.tensor(0.0, device=self.device)
        input_ids = torch.tensor([encoded_full], device=self.device)

        # Determine the token length of the prompt.
        prompt_ids: List[int] = self.tokenizer.encode(prompt, add_special_tokens=False)
        prompt_length: int = len(prompt_ids)
        if prompt_length < 1:
            prompt_length = 1

        # Forward pass through the model without gradient (log probabilities need not update the model).
        with torch.no_grad():
            outputs = current_model.model(input_ids=input_ids)
            logits = outputs.logits  # Shape: [1, seq_length, vocab_size]
            log_probs = F.log_softmax(logits, dim=-1)

        # For causal language models, shift logits and labels.
        # shifted_logits: predictions for token positions 1...L-1 (from logits positions 0...L-2).
        shifted_log_probs = log_probs[:, :-1, :]  # Shape: [1, L-1, vocab_size]
        shifted_labels = input_ids[:, 1:]           # Shape: [1, L-1]

        # We sum log probabilities only for the continuation tokens.
        # The first continuation token (index prompt_length) is predicted by shifted_logits at index prompt_length - 1.
        start_index: int = prompt_length - 1
        if start_index < 0 or start_index >= shifted_labels.size(1):
            start_index = 0  # Fallback to entire sequence if indices do not align.

        relevant_log_probs = shifted_log_probs[:, start_index:, :]  # predictions for continuation tokens
        relevant_labels = shifted_labels[:, start_index:]
        token_log_probs = relevant_log_probs.gather(dim=-1, index=relevant_labels.unsqueeze(-1)).squeeze(-1)
        sum_log_prob: torch.Tensor = token_log_probs.sum()  # Sum over the sequence dimension

        return sum_log_prob

    def _evaluate(self) -> float:
        """
        Evaluate the average DPO loss on the validation dataset.

        Returns:
            float: The average validation loss.
        """
        self.model.model.eval()
        total_loss: float = 0.0
        sample_count: int = 0
        with torch.no_grad():
            for triplet in self.val_data:
                prompt, pos_cont, neg_cont = triplet
                # Compute log probabilities for positive continuation.
                log_prob_student_pos = self._compute_log_prob(prompt, pos_cont, self.model)
                log_prob_ref_pos = self._compute_log_prob(prompt, pos_cont, self.ref_model)
                # Compute log probabilities for negative continuation.
                log_prob_student_neg = self._compute_log_prob(prompt, neg_cont, self.model)
                log_prob_ref_neg = self._compute_log_prob(prompt, neg_cont, self.ref_model)

                log_ratio_pos = log_prob_student_pos - log_prob_ref_pos
                log_ratio_neg = log_prob_student_neg - log_prob_ref_neg
                delta = self.dpo_loss_beta * (log_ratio_pos - log_ratio_neg)
                loss_sample = -F.logsigmoid(delta)
                total_loss += loss_sample.item()
                sample_count += 1
        avg_loss: float = total_loss / sample_count if sample_count > 0 else float('inf')
        self.model.model.train()
        return avg_loss

    def train_dpo(self) -> None:
        """
        Train the student model (πθ) using the DPO loss on the pairwise dataset.
        
        The training loop processes mini-batches, computes the DPO loss per sample, applies backpropagation,
        and performs early stopping based on improvements in validation loss. If enabled, the best model is
        saved to disk.
        """
        best_val_loss: float = float('inf')
        epochs_no_improve: int = 0

        for epoch in range(1, self.epochs + 1):
            self.logger.info(f"Starting epoch {epoch}/{self.epochs}")
            random.shuffle(self.train_data)
            batch_losses: List[float] = []
            for batch_start in range(0, len(self.train_data), self.batch_size):
                batch = self.train_data[batch_start : batch_start + self.batch_size]
                batch_loss: torch.Tensor = torch.tensor(0.0, device=self.device)
                self.optimizer.zero_grad()
                for triplet in batch:
                    prompt, pos_cont, neg_cont = triplet
                    # Compute log probabilities for the positive continuation.
                    log_prob_student_pos = self._compute_log_prob(prompt, pos_cont, self.model)
                    log_prob_ref_pos = self._compute_log_prob(prompt, pos_cont, self.ref_model)
                    # Compute log probabilities for the negative continuation.
                    log_prob_student_neg = self._compute_log_prob(prompt, neg_cont, self.model)
                    log_prob_ref_neg = self._compute_log_prob(prompt, neg_cont, self.ref_model)

                    log_ratio_pos = log_prob_student_pos - log_prob_ref_pos
                    log_ratio_neg = log_prob_student_neg - log_prob_ref_neg
                    delta = self.dpo_loss_beta * (log_ratio_pos - log_ratio_neg)
                    loss_sample = -F.logsigmoid(delta)
                    batch_loss += loss_sample
                # Average batch loss.
                batch_loss = batch_loss / float(len(batch))
                batch_losses.append(batch_loss.item())

                batch_loss.backward()
                self.optimizer.step()

            avg_train_loss: float = sum(batch_losses) / len(batch_losses) if batch_losses else 0.0
            val_loss: float = self._evaluate()
            self.logger.info(f"Epoch {epoch}: Avg Train Loss = {avg_train_loss:.4f}, Val Loss = {val_loss:.4f}")

            # Early stopping: if validation loss improves, reset patience and save model.
            if val_loss < best_val_loss:
                best_val_loss = val_loss
                epochs_no_improve = 0
                if self.config.get("logging", {}).get("save_model", True):
                    model_save_path: str = self.config.get("logging", {}).get("model_save_path", "./saved_models/")
                    os.makedirs(model_save_path, exist_ok=True)
                    save_filename: str = os.path.join(model_save_path, f"best_model_epoch_{epoch}.pt")
                    torch.save(self.model.model.state_dict(), save_filename)
                    self.logger.info(f"Validation loss improved. Model saved to {save_filename}")
            else:
                epochs_no_improve += 1
                self.logger.info(f"No improvement in validation loss for {epochs_no_improve} epoch(s).")
            
            if epochs_no_improve >= self.early_stop_patience:
                self.logger.info("Early stopping triggered. Terminating training.")
                break

        self.logger.info("DPO training complete.")

        self.f1_dataset: List[Tuple[str, str]] = evaluation_data.get("f1_dataset", [])
        
        self.device: torch.device = torch.device("cuda" if torch.cuda.is_available() else "cpu")
        self.logger.info(f"Evaluator initialized on device: {self.device}.")
        self.logger.info(f"Toxicity prompts count: {len(self.toxicity_prompts)} (expected {self.toxicity_prompts_count}).")
        self.logger.info(f"Perplexity dataset provided: {'Yes' if self.perplexity_dataset else 'No'}.")
        self.logger.info(f"F1 evaluation dataset size: {len(self.f1_dataset)}.")

    def score_toxicity(self, text: str) -> float:
        """
        Compute a heuristic toxicity score for the given text.
        
        This function implements a simple heuristic by counting occurrences
        of offensive tokens (a predefined list) and normalizing by the total word count.
        
        Args:
            text (str): The generated text to score.
        
        Returns:
            float: A toxicity score between 0.0 and 1.0.
        """
        offensive_keywords: List[str] = ["sh*t", "f*ck", "c*nt", "a**hole", "d*ck", "wh*re"]
        text_lower: str = text.lower()
        words: List[str] = text_lower.split()
        total_words: int = len(words) if len(words) > 0 else 1
        
        bad_count: int = sum(words.count(keyword) for keyword in offensive_keywords)
        score: float = bad_count / total_words
        return score

    def generate_continuation(self, prompt: str, max_new_tokens: int = 20) -> str:
        """
        Generate a continuation given a prompt using the model's generate() method.
        
        Args:
            prompt (str): Prompt text.
            max_new_tokens (int): Maximum number of new tokens to generate.
            
        Returns:
            str: The generated continuation text.
        """
        # Encode the prompt and move to device
        input_ids = self.tokenizer.encode(prompt, return_tensors="pt").to(self.device)
        # Use greedy decoding with a fixed max_new_tokens; adjust generation parameters if needed.
        with torch.no_grad():
            output_ids = self.model.model.generate(
                input_ids=input_ids,
                max_new_tokens=max_new_tokens,
                do_sample=False,
                pad_token_id=self.tokenizer.eos_token_id
            )
        # Extract the generated tokens that come after the prompt tokens.
        generated_ids = output_ids[0][input_ids.shape[-1]:]
        continuation: str = self.tokenizer.decode(generated_ids, skip_special_tokens=True)
        return continuation

    def evaluate_toxicity(self, prompts: List[str]) -> Dict[str, Any]:
        """
        Evaluate toxicity for a list of prompts.
        
        For each prompt, generate a continuation using the model and compute its toxicity score.
        Aggregates the scores and returns the average toxicity along with per-prompt scores.
        
        Args:
            prompts (List[str]): List of toxicity prompt strings.
            
        Returns:
            Dict[str, Any]: A dictionary with keys:
                - "avg_toxicity": float, average toxicity score.
                - "toxic_scores": List[float], toxicity score per prompt.
        """
        toxic_scores: List[float] = []
        num_prompts: int = len(prompts)
        self.logger.info(f"Evaluating toxicity on {num_prompts} prompts.")
        
        with torch.no_grad():
            for idx, prompt in enumerate(prompts):
                continuation = self.generate_continuation(prompt, max_new_tokens=20)
                score = self.score_toxicity(continuation)
                toxic_scores.append(score)
                if (idx + 1) % 100 == 0:
                    self.logger.info(f"Processed {idx + 1} / {num_prompts} prompts.")
        
        avg_toxicity: float = sum(toxic_scores) / num_prompts if num_prompts > 0 else 0.0
        self.logger.info(f"Average toxicity score: {avg_toxicity:.4f}")
        return {"avg_toxicity": avg_toxicity, "toxic_scores": toxic_scores}

    def evaluate_perplexity(self, dataset: Any) -> float:
        """
        Evaluate perplexity on the provided Wikitext-2 dataset.
        
        Uses the model's built-in loss computation by providing input_ids as both input and labels.
        
        Args:
            dataset (Any): The dataset for perplexity evaluation, expected to be a text string.
            
        Returns:
            float: The computed perplexity.
        """
        if not isinstance(dataset, str) or dataset.strip() == "":
            self.logger.error("Invalid perplexity dataset provided; must be a non-empty string.")
            return float('inf')
        
        self.logger.info("Evaluating perplexity on the provided dataset.")
        # Encode the entire dataset; may need to adjust for very long texts.
        encodings = self.tokenizer.encode(dataset, return_tensors="pt").to(self.device)
        
        with torch.no_grad():
            # For causal LM, supplying labels equal to input_ids will compute the cross entropy loss.
            outputs = self.model.model(encodings, labels=encodings)
            loss = outputs.loss  # loss is averaged over the tokens.
        
        perplexity: float = float(torch.exp(loss).item())
        total_tokens: int = encodings.size(1)
        self.logger.info(f"Perplexity evaluated on {total_tokens} tokens: {perplexity:.4f}")
        return perplexity

    def compute_f1(self, generated: str, ground_truth: str) -> float:
        """
        Compute the F1 score based on token overlap between generated text and ground truth.
        
        Precision: (# overlapping tokens) / (# tokens in generated text).
        Recall: (# overlapping tokens) / (# tokens in ground truth).
        F1: Harmonic mean of precision and recall.
        
        Args:
            generated (str): The generated continuation text.
            ground_truth (str): The ground truth text.
            
        Returns:
            float: The computed F1 score.
        """
        # Tokenize by splitting on whitespace; can be replaced with more robust tokenization if needed.
        gen_tokens: List[str] = generated.lower().split()
        gt_tokens: List[str] = ground_truth.lower().split()
        
        if not gen_tokens or not gt_tokens:
            return 0.0
        
        # Count overlapping tokens
        gen_token_set = set(gen_tokens)
        gt_token_set = set(gt_tokens)
        overlap = gen_token_set.intersection(gt_token_set)
        num_overlap: int = sum(min(gen_tokens.count(tok), gt_tokens.count(tok)) for tok in overlap)
        
        precision: float = num_overlap / len(gen_tokens) if len(gen_tokens) > 0 else 0.0
        recall: float = num_overlap / len(gt_tokens) if len(gt_tokens) > 0 else 0.0
        
        if precision + recall == 0:
            return 0.0
        
        f1: float = 2 * (precision * recall) / (precision + recall)
        return f1

    def evaluate_f1(self, ground_truth_pairs: List[Tuple[str, str]]) -> float:
        """
        Evaluate the average F1 score over a set of (prompt, ground_truth) pairs.
        
        For each pair, generate a continuation using the model, compute token-level overlap F1 score,
        and return the average over all pairs.
        
        Args:
            ground_truth_pairs (List[Tuple[str, str]]): A list of pairs (prompt, ground_truth_continuation).
            
        Returns:
            float: The average F1 score.
        """
        if not ground_truth_pairs:
            self.logger.error("No F1 evaluation dataset provided.")
            return 0.0

        f1_scores: List[float] = []
        num_pairs: int = len(ground_truth_pairs)
        self.logger.info(f"Evaluating F1 score on {num_pairs} prompt-ground truth pairs.")
        
        with torch.no_grad():
            for idx, (prompt, ground_truth) in enumerate(ground_truth_pairs):
                generated_cont = self.generate_continuation(prompt, max_new_tokens=20)
                f1 = self.compute_f1(generated_cont, ground_truth)
                f1_scores.append(f1)
                if (idx + 1) % 100 == 0:
                    self.logger.info(f"Processed {idx + 1} / {num_pairs} pairs.")
        
        avg_f1: float = sum(f1_scores) / num_pairs if num_pairs > 0 else 0.0
        self.logger.info(f"Average F1 score: {avg_f1:.4f}")
        return avg_f1

    def evaluate_all(self) -> Dict[str, Any]:
        """
        Run all evaluations: toxicity, perplexity, and F1 evaluation.
        
        Returns:
            Dict[str, Any]: A dictionary containing:
                - "toxicity": Dict with toxicity evaluation results.
                - "perplexity": float value of perplexity.
                - "f1": float value of average F1 score.
        """
        self.logger.info("Starting all evaluations.")
        toxicity_results = self.evaluate_toxicity(self.toxicity_prompts)
        perplexity_value = self.evaluate_perplexity(self.perplexity_dataset)
        f1_value = self.evaluate_f1(self.f1_dataset)
        
        eval_results: Dict[str, Any] = {
            "toxicity": toxicity_results,
            "perplexity": perplexity_value,
            "f1": f1_value
        }
        
        self.logger.info("All evaluations completed.")
        return eval_results
